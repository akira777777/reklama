# reklama — Telegram-юзербот для рассылки рекламы

Одноразовый CLI-скрипт на Python + [Telethon](https://github.com/LonamiWebs/Telethon):
рассылает сообщение из `message.txt` (с медиафайлом из `media/`) по всем групповым
чатам аккаунта, где отправка возможна.

> ⚠️ **На свой риск.** Массовая рассылка от имени личного аккаунта нарушает правила
> Telegram и может привести к **бану аккаунта**. Скрипт применяет смягчения (только
> группы, большие случайные паузы, перерывы пакетами, окно активности, обработка
> FloodWait), но полной защиты не существует.

## Установка

```bash
python -m venv .venv
.venv\Scripts\activate          # Windows
pip install -r requirements.txt # или requirements-dev.txt для разработки
```

Получите `api_id` / `api_hash` на https://my.telegram.org → *API development tools*.

```bash
cp .env.example .env            # затем впишите TELEGRAM_API_ID / TELEGRAM_API_HASH
```

Положите медиафайл в `media/` (берётся первый файл) и отредактируйте `message.txt`.

## Использование

```bash
# 1. Посмотреть, куда пойдёт рассылка (без отправки):
python run.py --dry-run

# 2. Дымовой тест на одной группе:
python run.py --limit 1

# 3. Полная рассылка (с resume по progress.json):
python run.py

# Сбросить прогресс и начать заново:
python run.py --reset-progress
```

При первом запуске скрипт интерактивно запросит телефон, код из SMS и (если есть)
пароль 2FA. Файл сессии `<SESSION_NAME>.session` сохраняется рядом со скриптом.

## Настройка (`.env`, необязательно)

| Переменная | По умолчанию | Описание |
|---|---|---|
| `FORCE_DOCUMENT` | `false` | Всегда отправлять медиа как документ |
| `DELAY_MIN_SEC` / `DELAY_MAX_SEC` | `30` / `90` | Диапазон паузы между сообщениями |
| `BATCH_SIZE` | `50` | Размер пакета перед длинным перерывом |
| `BATCH_PAUSE_MIN_SEC` / `BATCH_PAUSE_MAX_SEC` | `300` / `900` | Перерыв между пакетами |
| `ACTIVE_HOURS` | (выкл.) | Окно активности, напр. `09:00-21:00` |
| `MEDIA_PATH` | первый файл из `media/` | Явный путь к медиа |

## Структура

```
config.py     настройки + дефолты + переопределение из .env
auth.py       сессия Telethon, интерактивный логин
dialogs.py    перебор диалогов, фильтрация (чистые функции)
sender.py     отправка сообщения/медиа, классификация ошибок
progress.py   чтение/запись progress.json, resume (чистые функции)
run.py        CLI (argparse), оркестрация запуска
```
